# CHERDAK TV — публикация на GitHub Pages

Коротко: проект уже содержит workflow для деплоя `dist` на GitHub Pages через GitHub Actions.

## Что было исправлено в этом архиве
- Папка была названа **.gitgub/workflowes** — из‑за этого GitHub Actions **не запускались**.
  Правильно: **.github/workflows** (обязательно именно так).

## Пошагово: как выложить на GitHub Pages
1. Создайте новый репозиторий на GitHub (например, `cherdak-tv`).
2. Загрузите содержимое этой папки в репозиторий и сделайте `git push` в ветку `main`.
3. На GitHub откройте **Settings → Pages**:
   - **Source**: выберите **GitHub Actions**.
4. Откройте вкладку **Actions** и дождитесь завершения workflow **Deploy to GitHub Pages**.
5. После успешного деплоя GitHub покажет URL страницы (обычно вида `https://<username>.github.io/cherdak-tv/`).

## Как вывести на телевизор
- Откройте получившийся URL в браузере на телевизоре (или на приставке/ноутбуке, подключенном к ТВ).
- Включите полноэкранный режим:
  - ПК: `F11`
  - Chrome/Edge: меню → «Во весь экран»
- Если хотите режим «киоск» на ПК: запустите Chrome так:
  `chrome --kiosk https://<username>.github.io/cherdak-tv/`

Если страница пустая или не грузятся картинки — проверьте, что URL содержит название репозитория и заканчивается на `/`.
